Add to Claude Desktop's config file:

``` json
{
    "mcpServers": {
        "simple": {
            "command": "uv",
            "args": [
                "--directory",
                "/Users/resonance/projects/subfrequencies/resonance/simple",
                "run",
                "main.py"
            ]
        }
    }
}
```
